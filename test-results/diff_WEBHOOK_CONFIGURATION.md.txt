+## FOR PRODUCTION (Green-API Setup - Temporary Adapter)
+
+This repo can run in **Green-API mode** (text-only) using a dedicated webhook endpoint.
+
+### Step 1: Get instance credentials from Green-API
+In the Green-API console, copy:
+- `idInstance`
+- `apiTokenInstance`
+
+### Step 2: Configure webhook in Green-API
+Set:
+
+```
+Webhook URL: https://investo-backend-v2.onrender.com/api/greenapi/webhook
+Webhook URL token: <RANDOM_SECRET_TOKEN>
+```
+
+Notes:
+- Use a **random secret token** (no spaces). Example: `investo_greenapi_webhook_9f3a...`
+- Green-API will send an `Authorization` header on webhook calls when the token is set.
+
+### Step 3: Set Environment Variables in Render
+
+```bash
+WHATSAPP_PROVIDER=greenapi
+
+GREENAPI_API_URL=https://api.green-api.com
+GREENAPI_ID_INSTANCE=<YOUR_GREENAPI_ID_INSTANCE>
+GREENAPI_API_TOKEN_INSTANCE=<YOUR_GREENAPI_API_TOKEN_INSTANCE>
+
+# Must match the token you set in Green-API webhook settings
+GREENAPI_WEBHOOK_URL_TOKEN=<RANDOM_SECRET_TOKEN>
+```
+
+### Step 4: Redeploy Backend
+After adding env vars, trigger a redeploy in Render.
